Pulse Width at Fourier Transform Limit


The following are some calculator that will help you calculate the Pulse Width at Fourier Transform Limit (in fs),by inputting the value of Central wavelength (in nm) and Half Width of Laser Spectrum (in nm).


The formula that will be used for Pulse Width at Fourier Transform Limit will be as followed:

Pulse Width = $\frac{A\lambda_0^2}{c \triangle \lambda}$


Pulse Shape Constant, A:

Gaussian: 0.441
Sech: 0.315
Lorentz: 0.142

Half Width of Laser Spectrum, $\triangle \lambda$;
